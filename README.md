# used-car-market-analysis
Used Car Market Analysis

Overview

This project analyzes used car market data to identify pricing trends and relationships between key factors such as mileage, year, and brand.

The focus is on Porsche compared to other automotive brands including BMW, Mercedes, and Toyota.

Dataset

Public dataset from Kaggle containing used car listings (2013–2014).

Tools

Python
Google Colab
Data visualization libraries

Key Findings

Porsche is a premium brand with higher average prices
Mileage significantly affects car price
Small differences in production year have limited impact on price compared to mileage
Porsche has the highest average price among compared brands

Visualizations

Line charts for trend analysis
Scatter plots for relationship analysis
